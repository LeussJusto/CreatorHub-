# CreatorHub — Frontend (Vite + React + TypeScript)

Scaffold inicial del frontend para CreatorHub.

Requisitos:
- Node.js LTS (16+ recommended)
- npm o pnpm/yarn

Instalación y desarrollo (PowerShell):

```powershell
cd FRONTEND
npm install
npm run dev
```

Variables de entorno (opcional):
- `VITE_API_URL` -> URL base de la API (ej. http://localhost:4000)

Scripts:
- `npm run dev` - arranca vite en modo desarrollo
- `npm run build` - genera `dist/` para producción
- `npm run preview` - vista previa del build
- `npm run typecheck` - comprueba tipos TypeScript (no emite archivos)

Notas:
- Ejecuta `npm install` para instalar dependencias antes de arrancar.
- Si quieres que configure eslint/ prettier o testing (Jest/ Vitest), dímelo y lo añado.
